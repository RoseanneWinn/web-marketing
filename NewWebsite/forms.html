FAQ

How do I create a form that includes data fields from my database?

After selecting Data Sources and Fields as you would in a standard report, click the Design Form button in the Fields tab. Next, design your form using the WYSIWYG interface while adding fields using smart tags. For example, you would type in [FieldName] where you wanted the data from the FieldName column to go.

Input:
Country: [ShipCountry]
Number of orders: [OrderID]

Output
Country: Finland
Number of orders: 22

Country: Brasil
Number of orders: 83

Country: USA
Number of orders: 122

How do I import existing reports into Izenda Forms?

Depending on the format they are originally in, existing reports may be imported by saving them to HTML by using Microsoft word or a PDF to HTML converter. Additionally you can cut and paste directly into Izenda Forms with some HTML-enabled applications.


How do I create Master-Detail Reports?

  1. Select Data Sources and Fields
  2. Click Design Form and Add Fields and Subreports in brackets
  3. Add Filters and Preview the Report.

How does PDF rendering work?

The Izenda Forms PDF engine leverages next-generation web standards to deliver pixel perfect document generation based on HTML and CSS. This facilitates a consistent look and feel between the screen and printed versions because both are powered by the same underlying technology. Additionally, CSS formats and styles are fully supported independent of the user's browser. You can add HTML in any part of the design experience or via the backend API. The HTML-powered forms engine eliminates having to create separate styles for PDF and the screen. Users accustomed to working with PDF documents can store, save and email them to suit their needs.
