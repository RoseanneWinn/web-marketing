Izenda Vision™ is an extensible modern, HTML-based visualization framework for big data visualization. Visualizations are designed in HTML technologies like javascript, html, webgl, css, svg and css3d and libraries like jquery and d3.

HOW IT WORKS

Each visualization takes input JSON data from Izenda's response server and uses it to render an animated visual output. Customers can even extend Izenda Vision by creating their own visualizations or adding open source and third party visualizations. Visualizations can be used in a single report or combined with other types of visualizations, reports and forms into a dashboard.


Visualizations can be in the report designer, at the script level or via a WYSIWYG Excel-like designer.


Visualizations are found in the /Vision subfolder of the Izenda installation. Users can select from the available visualizations and tweak the base HTML on a per-report basis. A simple web-based HTML IDE lets visualizations be tweaked without having to log onto the server to modify files.


When a visualization is executed, parameters like the source URL are provided via the querystring which the javascript can then use to pull data from the response server or external data source. Additional parameters may alter the style of the visualization.


Each visualization can also have a design mode triggerred by the querystring. This indicates that additional selectors should appear, which allow for data or stylistic selections.
