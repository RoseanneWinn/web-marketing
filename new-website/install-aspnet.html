ASP.NET Installation

If IIS 7 doesn't work, install ASP.NET

  1. Select Start, All Programs, Accessories.
    1. You will see Command Prompt.
    2. Right-click on Command Prompt and select Run As Administrator.
  2. A dialog box will appear that reads: Do you want to allow the following program to make changes to computer?
    1. Select YES.
    2. The Command Prompt box will appear.
  3. Type in the following: cd \Windows\Microsoft.NET\Framework64\v2.0.50727 and press Enter.
  4. Now type in aspnet_regiis -i (Make sure there is a space before you put in -i) and press Enter.

  5. ASP NET is installed
